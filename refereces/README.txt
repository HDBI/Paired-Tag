This folder includes the references used for analysis of mm10 data. For other version of reference genome, please make your own ref file with the same format.

cell_id_full.fa ## used to generate a bowtie index for demultiplexing cellular barcodes
mm10.RNA.txt ## ref file for generating cell-to-genes matrix with `reachtools bam2Mtx2` function
mm10.bin5k.txt ## ref file for generating cell-to-bins matrix with `reachtools bam2Mtx2` function
